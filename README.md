## how to use:
- create an issue


issue template:  
```
APP_NAME=
APP_URL=
APP_ICON=
```
example:  
```properties
APP_NAME=HowToLiveLonger
APP_URL=http://sucicada.cf/HowToLiveLonger/
APP_ICON=https://user-images.githubusercontent.com/33519267/167297958-2bc084d2-e9f1-4dd5-a167-64904cab4ef9.png
```
